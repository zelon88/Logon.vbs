ORIGINAL VERSION DATE: 11/25/2022

CURRENT VERSION DATE: 11/25/2022

VERSION: v1.0

DESCRIPTION: 
A simple script for logging when a user logs in or out of a domain computer.


PURPOSE: 
To enhance the visibility of user logon/logoff operations.


INSTALLATION INSTRUCTIONS: 
1. Copy the entire "Logon" folder into the "AutomationScripts" folder on SERVER (or any other network accesbible location).
2. Create User GPOs that run this script at logoff of any user.
